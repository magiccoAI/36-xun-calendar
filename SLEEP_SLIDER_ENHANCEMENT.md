# 睡眠时长记录组件重构说明

## 改进概述

将原本简单的数字输入框重构为功能丰富的双轴滑块交互组件，提供更直观、更精确的睡眠时间记录体验。

## 核心功能

### 🎯 双轴滑块设计
- **入睡时间滑块**：范围 18:00 - 24:00（前一天晚上）
- **起床时间滑块**：范围 00:00 - 12:00（当天早上）
- **步长设置**：15分钟精度，满足实际记录需求
- **跨昼夜支持**：正确处理跨天睡眠计算

### 🎨 视觉设计
- **清新风格**：圆角设计、柔和阴影、低饱和度色彩
- **渐变轨道**：蓝色到紫色渐变，匹配现有UI主题
- **双触点设计**：蓝色入睡触点，紫色起床触点
- **实时反馈**：滑动时实时更新显示

### 📊 数据展示
- **时间显示**：清晰显示入睡时间、起床时间、总睡眠时长
- **时长计算**：自动计算跨昼夜睡眠时长（如23:30到07:30 = 8.0小时）
- **质量反馈**：三个快捷按钮记录睡眠质量

### 🔧 交互特性
- **拖拽支持**：支持鼠标和触摸拖拽
- **实时计算**：滑动时即时显示睡眠时长变化
- **质量按钮**：😴 睡得真香、😐 普普通通、😫 没睡好
- **自动备注**：点击质量按钮自动添加到备注栏

## 技术实现

### 数据结构
```javascript
sleepData: {
    bedtime: "22:30",        // 入睡时间
    wakeUpTime: "07:00",     // 起床时间  
    totalHours: 8.5,         // 总睡眠时长
    quality: "excellent"     // 睡眠质量
}
```

### 核心算法
- **跨昼夜计算**：`duration = (1440 - bedtime + wakeuptime) / 60`
- **时间转换**：分钟数 ↔ 时间字符串转换
- **位置计算**：滑块值 ↔ 百分比位置转换

### 组件架构
- **SleepSlider 类**：独立的滑块组件
- **Modal 集成**：无缝集成到现有记录模态框
- **数据兼容**：同时支持新旧数据格式

## 使用方法

### 基础操作
1. **拖拽滑块**：直接拖拽蓝色和紫色触点调整时间
2. **精确调整**：点击轨道任意位置快速定位
3. **质量反馈**：点击底部按钮记录睡眠感受

### 数据流程
1. **记录时间** → 2. **计算时长** → 3. **选择质量** → 4. **自动备注** → 5. **保存数据**
