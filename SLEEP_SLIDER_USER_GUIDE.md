# 睡眠时间滑块使用指南

## 🎯 功能说明

新的睡眠时间滑块让您可以直观地记录睡眠时间，系统会自动计算睡眠时长。

## 🖱️ 操作方法

### 基础操作
1. **拖动左侧圆点** 🌙 - 调整您的**入睡时间**（18:00-24:00）
2. **拖动右侧圆点** 🌤 - 调整您的**起床时间**（00:00-12:00）
3. **查看睡眠时长** - 中间会自动显示计算出的总睡眠时长

### 快速定位
- **点击轨道任意位置** - 快速将滑块定位到指定时间
- **支持触摸** - 移动设备上可以用手指拖动

## 📊 时间范围

- **入睡时间**：18:00（昨天）→ 24:00（今天）
- **起床时间**：00:00 → 12:00（今天）
- **调整精度**：15分钟步长
- **跨昼夜支持**：自动处理从晚上到早上的睡眠计算

## 🎨 视觉提示

- **蓝色圆点**：入睡时间滑块
- **紫色圆点**：起床时间滑块
- **渐变轨道**：显示睡眠时间段
- **实时显示**：拖动时即时更新时间

## 😴 睡眠质量记录

在滑块下方，有三个快捷按钮：

### 🌙 睡得真香
- 点击后按钮变为蓝色背景
- 自动在备注中添加"睡眠质量：😴 睡得真香"

### 😐 普普通通  
- 点击后按钮变为灰色背景
- 自动在备注中添加"睡眠质量：😐 普普通通"

### 😫 没睡好
- 点击后按钮变为红色背景
- 自动在备注中添加"睡眠质量：😫 没睡好"

## 💡 使用技巧

### 常见睡眠时间示例
- **早睡早起**：22:00 → 06:00 = 8小时
- **正常作息**：23:00 → 07:00 = 8小时  
- **晚睡晚起**：00:00 → 08:00 = 8小时
- **午休补觉**：13:00 → 14:00 = 1小时

### 操作建议
1. **先调整入睡时间**：拖动蓝色圆点到您的实际入睡时间
2. **再调整起床时间**：拖动紫色圆点到您的起床时间
3. **确认睡眠时长**：检查中间显示的总时长是否合理
4. **选择睡眠质量**：点击最符合您感受的按钮
5. **完成记录**：填写其他信息后保存

## 🔧 数据保存

系统会保存三个值：
- `bedtime`：入睡时间（如："23:30"）
- `wakeUpTime`：起床时间（如："07:00"）
- `totalHours`：总睡眠时长（如：7.5）

## ⚠️ 注意事项

1. **跨昼夜计算**：系统会自动处理从晚上到早上的时间计算
2. **15分钟精度**：时间调整以15分钟为单位
3. **质量记录**：睡眠质量会自动添加到备注栏
4. **实时更新**：拖动时所有显示会即时更新
