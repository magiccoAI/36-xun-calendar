## Required Icons:
- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- apple-touch-icon.png
- favicon-16x16.png
- favicon-32x32.png

## Screenshot files:
- screenshot-desktop.png (1280x720)
- screenshot-mobile.png (390x844)

## Shortcut icons:
- shortcut-today.png (96x96)
- shortcut-overview.png (96x96)

## Windows tile icons:
- icon-70x70.png
- icon-150x150.png
- icon-310x310.png
- icon-310x150.png

These can be generated from the main SVG favicon using tools like:
- https://favicon.io/
- https://realfavicongenerator.net/
- ImageMagick or similar tools
